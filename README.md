PinguiProfessions fundamentally transforms how players interact with the economy and their role in the server. Players enroll in immersive courses, complete interactive missions in specific zones, and earn their titles. 

## ✨ Features
*   **Interactive Courses**: Players walk into specific WorldGuard regions to start holographic courses.
*   **Unique Roles**: Features Doctors, Blacksmiths, Alchemists, Thieves, Investigators, Tavern Keepers, and Merchants.
*   **Clan Limits (SimpleClans)**: Configurable limits ensure clans cannot hoard all roles (e.g., max 2 Doctors per clan).
*   **Custom Mechanics**: 
    *   *Thieves*: Smoke bombs and stealth mechanics.
    *   *Investigators*: Analyzing player fingerprints and decrypting evidence.
    *   *Doctors*: Custom MedKits to heal other players.
*   **XP System**: Dedicated progression for each profession to unlock higher-tier perks.

## 📥 Compatibility
Compiled for **Java 21**. Compatible with:
*   Paper / Purpur 1.21+

**Dependencies:**
*   `WorldGuard` & `WorldEdit` (Required for course zones)
*   `DecentHolograms` (Required for interactive courses)
*   `SimpleClans` (Optional - for clan limits)

## 🛠️ Compiling from Source

To build from source:

```bash
git clone https://github.com/Angelguardian12/PinguiProfessions.git
cd PinguiProfessions
./gradlew build
```
You will find your JAR in `build/libs/`.
